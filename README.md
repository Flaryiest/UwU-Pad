# UwUPad
UwUPad is a 10 key macropad with 4 LEDs.  
This was my first time both CADing and designing a PCB and was a lot of fun!  

## Features 
4 SK6812Minis for RBG Lighting throughout the case  
10 Keys  
QMK Firmware
